Open-Source 2-Wheel Vehicle-Physics for Unity3D


Tested with version 2019.2.8

IntroductionLast updated: 2/2/2020

About

D.P.S. provides balance, suspension, engine, transmission, camera and sound systems for two-wheel vehicles.

Effortlessly turn models into fully functioning motorcycles in minutes.

How to use

Setting-up a vehicle
  1. Start with selecting a dirt bike, street bike or chopper prefab from the vehicles > prefabs folder
    Select the motorcycle prefab that most closly resembles your intented motorcycle
  2. Don't use the original prefab. Instead, duplicate the prefab so you have a template to reference
    From the hierarchy, select the prefab > Unpack prefab completely
  3. Save this duplicated object as a new prefab at any time by dragging it into your projects assets folder
  4. Values that should never change: Mass between 0.5 and 10, collision detection mode set to discrete, and vehicle scale one on all axes. To scale a vehicle, instead adjust it's scale on import, then move suspension parts to the correct positions.
